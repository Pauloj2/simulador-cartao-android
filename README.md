# 📱 Simulador de Cartão de Crédito – Android Kotlin
---
## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um aplicativo Android que simula a interface de um cartão de crédito com atualização dinâmica em tempo real.
O projeto foi desenvolvido seguindo duas etapas principais:
1. Criação do protótipo visual no **Canva**
2. Implementação do sistema no **Android Studio** utilizando **Kotlin**
O objetivo foi aplicar conceitos de:
- UI/UX
- ConstraintLayout
- Animações
- Manipulação de eventos
- Lógica de identificação de bandeira
---
## 🎨 Protótipo no Canva
Antes da implementação no Android Studio, foi criado um protótipo visual no Canva para definir:
- Estrutura da tela
- Distribuição dos elementos

- Experiência do usuário

## 🚀 Sistema Desenvolvido
Após o protótipo, o aplicativo foi implementado com:
- Kotlin
- ConstraintLayout
- Material Design Components
- Vector Drawables
- Shape Drawables
- Animação de rotação (flip)
---
## 📱 Funcionalidades Implementadas
- ✔️ Cartão com frente e verso
- ✔️ Animação de flip ao clicar no campo CVV
- ✔️ Atualização em tempo real dos dados digitados
- ✔️ Máscara automática para número do cartão
- ✔️ Máscara automática para validade (MM/AA)
- ✔️ Identificação dinâmica de bandeira (Visa, Mastercard, Outra)
- ✔️ Alteração automática de cor conforme a bandeira

## 🧠 Desafios Técnicos Trabalhados
### 🔹 UI Dinâmica
- Implementação de animação para alternar entre frente e verso do cartão utilizando rotação no eixo Y.
### 🔹 Lógica de Algoritmo
